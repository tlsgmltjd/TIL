### clone,commit,push 한 과정

1. 먼저 깃허브 레포 링크를 복사한후 로컬 저장소에 복제한다.

```ts
$ git clone "링크"
```

2. 디렉토리를 만들어 `terminal-command.md` `git.md` 과제 파일을 추가한다.

3. git add 를 사용하여 스태이징 한다.

```ts
$ git add .
```

4. git commit 를 사용해서 커밋한다.

```ts
$ git commit -m "커밋"
```

5. git push 를 사용해서 로컬 저장소의 변경사항을 원격 저장소로 업로드 한다.

```ts
$ git push
```
